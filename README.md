# Convite de Casamento - Jessica & Bruno 💍

Site de convite digital para o casamento de Jessica e Bruno, em 08 de Agosto de 2026.

## Estrutura

- `index.html` — página principal do convite
- `css/style.css` — estilos do site
- `js/script.js` — interatividade (menu, contagem regressiva, animações)
- `assets/` — imagens e ilustrações

## Seções

- Início (hero com contagem regressiva)
- Nossa História
- Cerimônia
- RSVP (confirmação de presença)
- Presentes

## Como visualizar

Basta abrir o arquivo `index.html` em um navegador, ou servir a pasta com um servidor local, por exemplo:

```bash
npx serve .
```
